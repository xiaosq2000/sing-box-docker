使用方法

下载后解压。

Windows 用户：运行 windows-amd64/start_proxy.bat 开启。（真不是 sing-box.exe）

Android 用户：安装 android-arm64/SFA-${VERSION}-universal.apk，导入配置文件 android-arm64/trojan-tun-client.json。

IOS/MacOS 用户：美区 AppStore 下载 sing-box，导入配置文件 android-arm64/trojan-tun-client.json。

Linux 用户：阅读 linux-amd64/README-LINUX.md。
